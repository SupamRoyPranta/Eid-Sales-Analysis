# Eid-Sales-Analysis
This project explores and analyzes Eid festival sales data to uncover customer behavior, spending patterns, and product performance. The goal is to gain business insights using Python-based data analysis and visualization techniques.

** Project Overview**

The Eid Sales Analysis project performs exploratory data analysis (EDA) on sales data to identify:

Which gender and age groups contributed most to sales

Top-selling product categories during Eid

Regional sales trends and performance insights

**Tools & Libraries Used**

Python

Pandas – Data cleaning and manipulation

NumPy – Numerical computations

Matplotlib & Seaborn – Data visualization

**⚙️ How to Run the Project**

Clone or download this repository

Open the .ipynb file in Jupyter Notebook or VS Code

Install the dependencies using:

pip install pandas numpy matplotlib seaborn

Run each cell sequentially to reproduce the analysis.

**Key Insights**

Majority of sales came from female customers

Age group 26–35 years made the highest purchases

Top categories: Clothing and Electronics

High-performing states contributed significantly to total sales
