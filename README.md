# Eid-Sales-Analysis
This project explores and analyzes Eid festival sales data to uncover customer behavior, spending patterns, and product performance. The goal is to gain business insights using Python-based data analysis and visualization techniques.

** Project Overview**

The Eid Sales Analysis project performs exploratory data analysis (EDA) on sales data to identify:

Which gender and age groups contributed most to sales

Top-selling product categories during Eid

Regional sales trends and performance insights

**⚙️Tools & Libraries Used**

Python

Pandas – Data cleaning and manipulation

NumPy – Numerical computations

Matplotlib & Seaborn – Data visualization


✔**Key Insights**

1. Majority of sales came from female customers

2. Age group 26–35 years made the highest purchases

3. Top categories: Clothing and Electronics

4. High-performing states contributed significantly to total sales
